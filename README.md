:calendar: Schedule
===
:large_blue_diamond: 개요
---
일정 관리 앱 서버 프로젝트를 작성하였습니다.

API 명세서와 ERD를 만들고 이를 바탕으로 Spring과 SQL을 통해 DB와 CRUD를 구현합니다.

**개발 환경**
Oracle JDK 17.0.14
IDE: IntelliJ
Framework: Spring
